# Student Activity Tracker

Students need a trusted, centralized, and student-only marketplace that makes it easier to sell used items and buy affordable, good-quality furniture.

## Installation & Setup

 **Install Dependencies**
    Run the following commands to install the necessary libraries, including the icon pack:
    ```bash
    npm install
    npm install react-icons
    ```

## How to Run

**Step 1: Start the Frontend Open a new terminal window inside the project folder and run:**
```
npm run dev
```
**Step 2: Open in Browser**
- Look at the "Step 2" terminal output.
- Control-click the link shown (usually http://localhost:5173 or http://localhost:5174).
